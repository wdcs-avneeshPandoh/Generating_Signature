# Generating_Signature








steps to run :-
1) install thunder client plugin
2) enter the url : http://localhost:9000/signing_msg   .
3) change the command to post.
4) enter the message into body (Json format).
5) message example:
                          {
                            "message": "0xac5356c0587d15ecfd2601666e232ef0d6fa5bf925e4025354d6ee7e05266d8d"
                          }
6) the message is generated.
7) you can change the public keys of specified guardians in .env file.
